## Overview

**Huashu's Content Creation Skills Collection — 21 Practical Skills including AI Proofreading, Topic Generation, Video Outlining, and Material Search.**

This repository contains a curated collection of highly optimised, professional AI agent skills specifically designed for content creators. These 21 battle-tested practical skills cover the complete creative workflow, handling a wide range of tasks from writing and social media management to design, presentations, and video planning. Each skill is structured as a standard Markdown file (`SKILL.md`) that can be seamlessly loaded into compatible AI agents to equip them with specialised, expert-level capabilities.

## How to Use

To use the skills from this repository in your AI agent environment (like ChatGPT, Claude, or any custom agentic platform), you'll essentially be "loading" the specific skill instructions into the agent.

Here is how you can use them:

1. **Clone the Repository:** First, download the repository to your local machine:

   ```bash
   git clone https://github.com/Biraj2004/huashu-skills-english.git
   ```

2. **Pick a Skill:** Browse the repository and choose a skill folder that matches your task (e.g., `huashu-data-pro-en` for data analysis or `huashu-xhs-image-en` for generating Xiaohongshu images).
3. **Open the `SKILL.md` File:** Inside your chosen folder, you'll find a file named `SKILL.md`. This file contains the complete persona, system prompt, rules, and structured workflow for that specific skill.
4. **Feed it to Your Agent:**
   * **Direct Copy/Paste:** The simplest method is to copy the entire contents of the `SKILL.md` file and paste it directly into your chat with the AI (or into its "System Prompt" / "Custom Instructions" section if the platform supports it).
   * **File Upload/Reference:** If your AI tool supports file attachments or workspace reading, simply attach the `SKILL.md` file and tell the AI: *"Please read this `SKILL.md` file and adopt its workflow for my next requests."*
5. **Follow the Workflow:** Once loaded, the AI will adopt the specialized persona and guide you through the process as defined by the creator, ensuring high-quality, professional outputs!
