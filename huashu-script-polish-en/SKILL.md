# Video Script Colloquialisation

Make video scripts more suitable for actual recording: strengthen the spoken feel, remove formal written language and AI-sounding tone.

## When to Use

- After a script draft is complete
- When the user says "proofread the script" or "too formal/written"
- When the script needs to sound more conversational

## Core Objective

- Remove formal written language — fully colloquialise
- Suitable for reading aloud (no stumbling)
- Sounds like Huashu speaking to camera

## Three-Pass Review (Video-Specific)

### Pass 1: Content Review

- [ ] Facts accurate, demonstrations feasible
- [ ] Logic clear, steps coherent
- [ ] Length appropriate (150–180 words per minute)

### Pass 2: Colloquialisation Review (Core Pass)

**Remove formal language**:

- ❌ "Next, we will proceed to..." → ✅ "Let's..."
- ❌ "Perform the operation" → ✅ "Click"
- ❌ "Implement the functionality" → ✅ "Get it done"
- ❌ "In summary" → ✅ Just summarise directly

**Add conversational words**:

- ✅ "Actually", "Look here", "Right here", "So then"
- ✅ "Honestly", "Earlier I...", "This is the key part"

**Make it sayable**:

- Short sentences (no more than 20 words)
- Use full stops to break sentences, not commas to join long ones
- Read it aloud — rewrite anything that causes you to stumble

### Pass 3: Recording Annotations

- [ ] Mark where to cut to screen, where to do close-ups
- [ ] Check pacing and pauses
- [ ] Simulate the actual recording flow

## Colloquialisation Rewriting Examples

```text
❌ Formal:
Next, we will proceed with the Claude Code installation. First, open the terminal, then enter the corresponding command...

✅ Colloquial:
Right, let's get Claude Code installed. Simple — open your terminal, type this command... (cut to screen) See? Just like that.
```
