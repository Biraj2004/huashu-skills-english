# Security Policy

## Supported Versions

Only the latest version of the skills (the current state of the main/master branch) is actively supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of this repository seriously. **Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within this repository, please report it via the following method:

1. **GitHub Private Vulnerability Reporting:** If enabled on this repository, please go to the **Security** tab, select **Advisories**, and click **Report a vulnerability**.
2. **Email:** Alternatively, please contact the maintainer directly to report the issue privately.

Please provide as much detailed information as possible about the vulnerability, including:

- A description of the vulnerability and its potential impact.
- Step-by-step instructions to reproduce the issue.
- Any suggested mitigations or fixes.

We will acknowledge receipt of your vulnerability report as soon as possible and work with you to address it. We kindly request that you do not publicly disclose the vulnerability until a fix has been issued.
