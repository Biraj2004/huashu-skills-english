# Video Outline Generation

Quickly generate 2–3 video script outline options, each including a title, thumbnail recommendation, and structural design.

## When to Use

- After saving a video brief
- When the user says "give me an outline" or "suggest a few options"
- Step 3 of the video creation workflow (outline discussion)

## Core Features

1. **Provide 2–3 outline options**: Each from a different angle
2. **Each option includes**: Title, thumbnail recommendation, core angle, outline structure, estimated duration
3. **Apply thumbnail strategy**: 5 strong contrast formulas + thumbnail design strategy

## Outline Option Template

```markdown
## Option 1: [Compelling Title]
**Thumbnail Recommendation**: Results-based display / Face shot (depends on subscriber count)
**Core Angle**: xxx
**Estimated Duration**: ⏱️ 5–8 minutes

### Outline Structure
1. Opening (30 seconds – 1 minute): Confirm title promise + Build suspense
2. Core Module 1 (2 minutes): xxx
3. Core Module 2 (2 minutes): xxx
4. Surprise Element (1 minute): Content that exceeds expectations
5. Closing (1–2 minutes): Deliver on the promise + Call to Action

### Demonstrations Required
- Demo 1: xxx
- Demo 2: xxx

### Pros and Cons
✅ Advantages: xxx
⚠️ Disadvantages: xxx
```

## Title Strategy (5 Types of Contrast)

1. Quantity contrast: "1 Feature vs 10 Tools"
2. Price contrast: "Free vs Paid — How Big Is the Gap?"
3. Results contrast: "0 to 300K Followers — What I Did Right"
4. Strength contrast: "Beginner vs Cursor — Who Wins?"
5. Time contrast: "3 Minutes vs 3 Hours"

## Thumbnail Strategy

- **< 100K subscribers**: Prioritise results-display thumbnails
- **> 100K subscribers**: Can test face thumbnails
- **Tutorial content**: Before/After comparison
- **Review content**: Product close-up + data

## Notes

- Apply content engagement strategy (Opening confirmation → Mid-section surprise → Closing payoff)
- Duration estimates should be realistic
- Wait for the user to choose — do not assume
